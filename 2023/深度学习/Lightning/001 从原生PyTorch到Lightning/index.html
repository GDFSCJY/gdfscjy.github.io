什么是Lightning

Lightning库是一个组织好的PyTorch代码库。
即使PyTorch本身足够简单易用，但是项目一旦庞大，代码仍然会变得凌乱复杂。

Lightning库出现是为了让开发者更专注于算法逻辑的实现，并且减少工程代码的复写。

安装

pip install lightning

一些调整Trainer的参数

Trainer的作用是控制LightningModule已设置好的逻辑进行，这意味着你不用再去写for循环控制训练和验证或者设置变量去记录指标，
它甚至可以自动把模型和输入数据放入到指定计算设备中，并且在验证模型时自动禁止计算梯度并切换模型到验证模式。

Epoch

训练次数（Epoch）是DL里很关键的一个参数，Trainer里包含两个关于Epoch的参数max_epochs和min_epochs

max_epochs是指一旦到达指定次数，则终止训练；min_epochs是指到达指定次数之前，训练是不会停止的。
显然max_epochs更符合原生PyTorch中num_epochs参数的意思，而min_epochs更常用于早停策略训练中。

计算设备

Lightning支持CPU，GPU，TPU，IPU设备，并且支持分布式策略

在Lightning 2.0后，Trainer支持自动设置加速设备
