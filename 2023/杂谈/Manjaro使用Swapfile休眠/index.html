Manjaro使用Swapfile休眠

发表于 2023-04-19 分类于 杂谈 阅读次数： 本文字数： 340 阅读时长 ≈ 1 分钟
个人更喜欢用Swapfile，十分滴灵活

设置交换文件Swapfile

见 Manjaro安装后配置

查询硬盘UUID

sudo lsblk -f

我的Swapfile存储在UUID=460e09bb-a3ac-47ad-846a-adc90a34b34d里

查询Swapfile偏移

sudo filefrag -v /swapfile

表格第一行的physical_offset列左边的数字就是偏移量，这里是16484352

修改grub文件

文件地址：/etc/default/grub

找到 GRUB_CMDLINE_LINUX_DEFAULT 项目，内容可能是：
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash udev.log_priority=3"

在后面添加内容resume=UUID resume_offset=Offset，即：
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash udev.log_priority=3 resume=UUID=460e09bb-a3ac-47ad-846a-adc90a34b34d resume_offset=16484352"

添加resume钩子

文件地址：/etc/mkinitcpio.conf

找到 HOOK 项目，内容可能是：
HOOKS="base udev autodetect modconf block keyboard keymap consolefont plymouth filesystems fsck"

在 udev 和 autodetect 之间添加 resume（不确定顺序是否有影响），即：
HOOKS="base udev resume autodetect modconf block keyboard keymap consolefont plymouth filesystems fsck"
