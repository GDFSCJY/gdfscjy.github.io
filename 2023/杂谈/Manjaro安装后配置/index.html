    Manjaro安装后配置

    发表于 2023-04-16 分类于 杂谈 阅读次数： 本文字数： 581 阅读时长 ≈ 2 分钟
    个人记录：）不得不说Manjaro+KDE比隔壁Ubuntu+GNOME清爽太多了

    换源

    自动换源

    sudo pacman-mirrors -i -c China -m rank

    更新

    更新系统 sudo pacman -Syu，
    但其实直接运行yay更加方便

    常用工具

    编译工具

    sudo pacman -S base-devel

    AUR

    sudo pacman -S yay

    deb包解压

    yay -S debtap

    微信

    yay -S com.qq.weixin.spark

    若微信的输入出现方框，需要安装字体

    yay -S wqy-microhei

    把字体复制到Wine环境

    cp /usr/share/fonts/wenquanyi/wqy-microhei/wqy-microhei.ttc /home/用户名/.deepinwine/Deepin-WeChat/drive_c/windows/Fonts/

    微信边框在失焦时仍然显示，需要设置wine winecfg

    在显示页面，去掉允许窗口管理器装饰窗口

    英伟达驱动

    sudo mhwd -a pci nonfree 0300

    HP打印机驱动

    yay -S hplip

    然后运行一次sudo hp-plugin

    交换文件Swapfile

    创建Swap文件
    sudo dd if=/dev/zero of=/swapfile bs=1M count=4096 status=progress

    设置权限，仅root用户可读写
    sudo chmod 600 /swapfile

    格式化并启用Swap文件

    在系统启动时自动装载Swap，在/etc/fstab添加入口
    sudo bash -c "echo /swapfile none swap defaults 0 0 >> /etc/fstab"

    设置Swap激进程度

    在 /etc/sysctl.conf 中添加参数：

    vm.swappiness=10

    交换的激进程度设置在0到100，数值越高越激进

    Anaconda和ZSH

    ZSH的配置文件是.zshrc，把在.bashrc中Anaconda部分复制到.zshrc并source .zshrc即可

    开机自动挂载盘

    查找要挂载的硬盘

    输入：
    sudo fdisk -l

    输出，找到要挂载的盘：

    查看UUID

    输入：
    ls -l /dev/disk/by-uuid

    输出，找到/dev/nvme1n1p1对应UUID为034A-9602：

    编辑/etc/fstab

    我设定的挂载点是/mnt/Cache，可自行设置

    在最后一行添加：
    UUID=034A-9602 /mnt/Cache exfat defaults 0 0

    我的硬盘文件系统是exFAT，因此设置为exfat；若为NTFS，设置为ntfs

    重启
