个人配置：）小孩子不懂事，建站建着玩的

安装Node.js

推荐安装LTS的

Windows从官网下载，同时记得装git

Manjaro用yay -S nodejs-lts npm

或者使用nvm版本管理器安装Node.js

安装Hexo

全局安装Hexo

npm install hexo -g

在博客目录初始化Hexo

就会生成Hexo博客项目啦，接下来往source/_post扔markdown

生成页面

hexo clean && hexo g

在public里生成了一个静态网站，扔到服务器上部署吧
